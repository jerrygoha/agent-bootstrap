## Notes

- README content helps both external web indexing and GitHub internal search.
- GitHub topics and the repository description matter more than classic meta tags because GitHub repositories do not expose normal SEO controls.
- Keep the canonical README in English, and keep translated README variants linked at the top of every translation.
